0x02 解决方法

在网站的源代码中，我们可以发现原来的html标签被处理成了这个样子。

也就是说，<ruby> <rt>等标签被处理成了<!-- raw HTML omitted -->。

一番搜索之后,在官方论坛找到了这个问题的产生原因和解决方法。

Raw HTML getting omitted in 0.60.0

原因就是，在19年11月的Hugo的0.60.0版本中， 使用了Goldmark作为默认的Markdown库，如果想再次使用HTML标签的话，需要在配置文件中添加以下的配置。 

[markup]
  [markup.goldmark]
    [markup.goldmark.renderer]
      unsafe = true

添加以上几行之后就可以在Hugo中愉快的使用HTML标签辣w。
